# SQL

## Common Basic Commands

### UPDATE Example

```
UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition;
```

### INSERT Example

```
INSERT INTO table_name
VALUES
(value1,value2,...);
```

## Altering a Table

### Drop Column Example

```
ALTER TABLE table_name
DROP COLUMN column_name;
```

### Add Column Example

```
ALTER TABLE Customers
ADD Email varchar(255);
```
